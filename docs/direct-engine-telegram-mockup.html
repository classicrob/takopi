Direct Mode vs Liaison Mode

Aspect Direct Liaison
Command style /claude, /codex, /pi, /opencode /liaison (natural language)
Execution Single agent runs directly Liaison orchestrates multiple agents
Progress view Tool calls only Subagent panes with live output
User interaction Send command, wait for result May receive escalation questions
Best for Quick, focused tasks Complex multi-step workflows
Resume token claude --resume xyz liaison --session xyz
