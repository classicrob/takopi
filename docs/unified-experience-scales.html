The Unifying Principle

Every scenario uses the same three components:

1. Session Card — shows what's happening (scales from 1 to N agents)
2. Activity Feed — shows the details (expand when curious)
3. Input Prompts — collect decisions (same buttons, same replies)

The user never thinks "am I in simple mode or complex mode?" They just talk to Takopi.
